# معلمي (Mualimi) 🎓
نظام إدارة تعلم (LMS) بسيط وجميل للمعلمين والطلاب.

## المميزات 🌟
- **لوحة تحكم للمعلم**: إدارة المجموعات، التحضير، وإنشاء الاختبارات.
- **لوحة تحكم للطالب**: متابعة الاختبارات النشطة، النتائج، وسجل الحضور.
- **تفاعلي بالكامل**: واجهة مستخدم حديثة وسريعة باستخدام Bootstrap 5.
- **مزامنة البيانات**: يعتمد على `localStorage` لحفظ البيانات محلياً.

## كيف تبدأ؟ 🚀
فقط افتح ملف `index.html` في متصفحك وابدأ الاستخدام!
